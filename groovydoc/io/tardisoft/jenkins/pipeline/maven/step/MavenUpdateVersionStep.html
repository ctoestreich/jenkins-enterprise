[Groovy] Class MavenUpdateVersionStep

      • io.tardisoft.jenkins.pipeline.maven.step.MavenUpdateVersionStep
      • All Implemented Interfaces and Traits:
        Serializable, Step

        class MavenUpdateVersionStep
extends MavenStep

        Called at the beginning of the build to "update" the pom.version This is used for changing the version to a release version, such as 1.1.0-SNAPSHOT changes to 1.1.0 or to update the version to something specific to the branch, such as 1.1.0-SNAPSHOT changes to 1.1.0.BRANCH-SNAPSHOT How the version is actually calculated is delegated to the ReleaseStrategy
