[Groovy] Class GitTagReleaseStrategy

      • io.tardisoft.jenkins.pipeline.release.GitTagReleaseStrategy
      • All Implemented Interfaces and Traits:
        ReleaseStrategy, Serializable

        class GitTagReleaseStrategy
extends BasicReleaseStrategy

        Extends from basic release strategy but calculates the release version from the git tags. For instance if the git repo has the tags 1.2.0 1.3.0 1.4.0 and the pom has version 1.3.X the new release will be 1.3.1
